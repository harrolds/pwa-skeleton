# PWA config template (Skeleton V3)

Gebruik dit als referentie om van de skeleton naar een nieuwe PWA te gaan. Pas alles aan binnen je eigen repo; deze map wordt niet door de build meegenomen.

## Stappenplan (kort)

1. Clone deze skeleton of gebruik hem als template voor je nieuwe repo.
2. Pas `package.json` naam/description aan (alleen in je nieuwe project).
3. Vul `src/config/appConfig.ts` in aan de hand van `appConfig.example.ts` (branding, kleuren, logo).
4. Werk i18n bij: gebruik `i18n-example.en.json` / `i18n-example.de.json` als minimale basis voor `src/locales/<lang>.json`.
5. Activeer modules in `src/config/moduleRegistry.ts` en pas navigatie-labels aan (i18n).
6. Controleer icons/branding in `public/` en PWA-manifestwaarden.

Zie `docs/GettingStartedNewPWA_V3.md` voor het volledige stappenplan en checklist.
