- React 18 + Vite + TypeScript
- Repo: pwa-skeleton (Git is enige SSOT)

SSOT VOOR DEZE SESSIE:
- Bijgevoegde ZIP: pwa-skeleton_fase_*.zip  ← dit is de enige codebron voor deze fase
- Bevat o.a.:
  - SKELETON_V1_SCOPE.md
  - Skeleton_V1_Bouwpad.md
  - volledige huidige codebase t/m vorige fase

HUIDIGE FASE:
- Fase X  ← vul hier het fasenummer in, bv. "Fase 2"

HARDE REGELS VOOR DEZE SESSIE:
1) Gebruik uitsluitend de inhoud van de bijgevoegde ZIP als basis; geen aannames buiten deze ZIP.
2) Houd je strikt aan SKELETON_V1_SCOPE.md en Skeleton_V1_Bouwpad.md in de ZIP; werk alleen binnen de genoemde fase.
3) Respecteer de single root policy: `src/main.tsx` rendert uitsluitend `AppRoot` uit `src/core`; maak geen extra top-level rootcomponenten.
4) Introduceer geen nieuwe patronen, structuren, checklists of templates.
5) Geef geen samenvattingen of analyses; lever alleen de gevraagde bestanden.

OPDRACHT VOOR DEZE SESSIE:
- Open en inventariseer de bijgevoegde ZIP als actuele baseline.
- Voer Fase X uit zoals beschreven in Skeleton_V1_Bouwpad.md, op basis van deze baseline.
- Lever alle benodigde complete bestanden voor deze fase.
- Output uitsluitend:
  - het volledige pad van elk aangepast/toegevoegd bestand,
  - gevolgd door de volledige file-inhoud.
- Geen extra uitleg, geen tekst buiten pad + file-inhoud.
