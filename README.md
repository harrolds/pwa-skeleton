# PWA Skeleton v1

Dit is het **Skeleton v1** voor de PWA Factory: één herbruikbare basis voor mobiele/tablet PWA’s in portrait mode, gebouwd met React 18 + Vite + TypeScript.

- Doel: een stabiel fundament om PWA’s als Dreamdrop, Temio en RiskRadar snel en consistent op te zetten.
- Primary targets: **mobiel en tablet (portrait)**. Desktop toont alleen een blokkerscherm met QR-code.

---

## Stack

- React 18
- Vite
- TypeScript
- Client-side routing
- PWA (manifest + service worker)
